# Bài tập lớn môn Công Nghệ Phần Mềm

## Giới thiệu sơ lược

### Thông tin cơ bản
* Lớp: 75DCTT21
* Đề bài: Quản Lý Hồ Sơ Sinh Viên

### Thành viên thực hiện
1. Phạm Trung Hiếu *(A.K.A **Pipo**)*
2. Phạm Văn Đam *(A.K.A **P.D Dante**)*

### Các công nghệ chính sử dụng (Tech Stack):
* **Ngôn ngữ lập trình:** *C++* (C++17)
* **Khung phát triển và giao diện (Framework/UI):** *Qt 6* (Qt Widgets, Qt Designer)
* **Cơ sở dữ liệu:** *SQLite* (QSQLITE via QSqlDatabase)
* **Kiến trúc phần mềm (Architecture):** *MVC* (Model-View-Controller)
* **Hệ thống xây dựng (Build System):** *CMake*
